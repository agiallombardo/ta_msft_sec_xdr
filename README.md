
## Logging

- Logs are stored in `$SPLUNK_HOME/var/log/splunk/ta_msft_sec_xdr_*.log`
- Set the log level in the add-on settings (Configuration > Logging)
- For detailed troubleshooting, set the log level to DEBUG
